About me

My research interest includes natural language processing, machine learning, deep learning and data mining. Since my undergraduate major is Mathematics and Applied Mathematics, I am particularly interested in how the statistical method can be applied in these three fields. Currently, I am focusing more on how deep learning tools can be used to solved problems in natural language processing. Previously, I obtained my bachelor degree in Mathematics and Applied Mathematics from Shenzhen University. And I will graduate in Apr 2019 from University of Pittsburgh.

Identify Native Language of non-native Writer

Build native language identification (NLI) system
Logistic regression with different penalty is used to do feature selection and classification.
Search for the best hyperparameter using GridSearch with 5-fold cross-validation.
Precision, recall and F1-score on test set are 69%.

Distinguish hate speech from offensive language

Clean the data collected by making use of Twitter API.
Use word-based TF-IDF and POS-based TF-IDF to achieve a very good baseline.
Select various features using logistic regression.
SVM, logistic regression and deep learning techniques are used to classify tweets.
Obtain 89% accuracy which is very close to the start-of-the-art.

Classify Toxic Comments Using Keras and Bidirectional LSTM
